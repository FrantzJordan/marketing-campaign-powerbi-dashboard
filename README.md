## Overview
This project analyzes marketing campaign performance using a star-schema model in Power BI.

It includes KPIs such as CTR, Conversion Rate, ROI, Acquisition Cost, and interactive drill-down reporting.

## Data Model
- MarketingCampaign (Fact)
- Channel (Dim)
- Audience (Dim)
- CampaignType (Dim)
- Date (Dim)

## Key Metrics (DAX)
- CTR = Clicks / Impressions
- Avg Conversion Rate
- Cost per Click (CPC)
- ROI Performance

## Dashboard Pages
### Executive Overview
High-level campaign KPIs and performance breakdown.

### Campaign Drill-through
Interactive campaign-level exploration by company and channel.

### Data Quality Monitoring
Tracks missing values, duplicates, and reporting reliability.

## Tools Used
- Power BI Desktop
- Power Query (ETL)
- DAX Measures
- Star Schema Modeling
